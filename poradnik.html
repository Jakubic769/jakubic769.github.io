Poradnik: Jak stworzyć stronę WWW (tak jak ja :))

1. Utwórz pliki HTML, CSS i JS.
2. Zapisz je w jednym folderze.
3. Otwórz plik `index.html` w przeglądarce.
4. Opcjonalnie: wgraj stronę na GitHub Pages lub Netlify.
5. Gratulacje! Masz swoją pierwszą stronę.
